# መሣሪያዎች እና አፈጣጠሮች


እንኳን ደህና መጡ ፣ እንደ አስፈላጊ የድር አዘጋጆች! በዚህ ርዕስ ውስጥ የድር ልማት አካሄዳችንን ለማቀናጀታችን በጣም አስፈላጊ በሆነው መዋቅር ውስጥ እንገባሃለን ። ወደ ድር ልማት ዓለም ለሚደረግ የተቀላጠፈ ጉዞ ትክክለኛ መሳሪያዎችን መረዳትና ማዋቀር ወሳኝ ነው ። ዝርዝር ማብራሪያዎችን በመስጠት የሚያስፈልጓቸውን ቁልፍ የሆኑ መሣሪያዎችና ዘዴዎች እስቲ እንመልከት ።

### የድር አሳሽ
የመጀመሪያው መሳሪያችን የድር አሳሽ ነው ፣ የምትፈጥሯቸውን የድር ገጾች ለማሳየት የምንጠቀምበት በር ነው ። የድር አሳሽ በአለም ውደ ድር ላይ መረጃን ሰርስሮ ለማውጣት እና ለማቅረብ የተዘጋጀ የሶፍትዌር መተግበሪያ ነው ። ተወዳጅ ከሆኑት አሳሾች መካከል ጎግል ክሮም ፣ ሞዚላ ፍሪፎክስ እና ማይክሮሶፍት ኤድጌ ይገኙበታል ። ለዚህ አካሄድ ፣ የ Google Chrome ዓይነት እንዲጠቀሙ እመክራለው ፣ ነገር ግን በጣም የሚመችዎትን አሳሽ ለመምረጥ ነፃነት ይሰማኛል ። ክሮም የተጫነ ከሌለዎት ፣ ከዚህ በታች ባለው መግለጫ ውስጥ የማውረድ አገናኙን ማግኘት ይችላሉ ።

### የጽሁፍ አርታዒ
ቀጥሎ ያለው የጽሁፍ አርታኢ ፣ HTML ፣ CSS እና JavaScript ፋይሎችን ለመጻፍ የሚያስችል መሠረታዊ መሳሪያ ነው ። የጽሁፍ አርታዒ ግልጽ የጽሁፍ ፋይሎችን ለመፍጠር እና አርትዕ ለማድረግ የሚያስችል የሶፍትዌር መተግበሪያ ነው ። ለሚሰጠን ሥልጠና Visual Studio ኮድን መጠቀምን በእጅጉ እመክራለሁ ። እንደ ኖትፓድ ፣ Visual Studio ኮድ ከመሳሰሉት መሰረታዊ የፅሁፍ አርታኢዎች በተለየ መልኩ ለድር ልማት በቀጥታ የተስተካከለ ሃይለኛ ፣ ነጻ እና ግልፅ አዘጋጅ ነው ። ኮዴክሱን ይበልጥ ቀልጣፋና አስደሳች ከሚያደርጉት ገጽታዎች ጋር የሚመጣጠን ነው ። በ Visual Studio ኮድ ለማውረድ እና የኮድዎን ተሞክሮ ለማሳደግ በዚህ መግለጫ ውስጥ ወዳለው አገናኝ ያቀኑ ።

### ጊታ እና ጊትሃብ
በመሳሪያዎቻችን ውስጥ ያለው ሦስተኛው መሣሪያ ጊት ፣ የስሪት ቁጥጥር ሥርዓት እና ጊትሁብ በኮድ ላይ የሚከማችበት እና በጋራ የምንሰራበት መድረክ ነው ። እስቲ የሚከተሉትን ነጥቦች እንመርምር ፦

**ሁለገብ ቁጥጥር ምንድን ነው?**
የስሪት ቁጥጥር በፋይል ላይ የተደረጉ ለውጦችን ወይም የፋይሎችን ስብስብ በጊዜ ሂደት የሚመዘግብ ስርዓት ነው ፣ ስለዚህ የተወሰኑ ስሪቶችን በኋላ ማስታወስ ይችላሉ ። ትብብርን ያመቻቻል እናም ፋይሎችን ወደ ቀደመ ሁኔታ ዳግም እንዲመልሱ ይፈቅድልዎታል ።

**ጊት - ሁለገብ ቁጥጥር ሥርዓት፦**
ጊት በስፋት የተሰራጨ የስሪት ቁጥጥር ስርዓት ነው ። በኮድዎ ውስጥ የተደረጉ ለውጦችን ለመከታተል ፣ ከሌሎች ጋር በጋራ ለመስራት ፣ እና የፕሮጀክትዎን የተለያዩ ስሪቶች ያለገደብ ለማስተዳደር ይፈቅድልዎታል ።

**ጊታሩብ - በጋራ መሥራቴ ንድፍ**
ጊትሁብ ለሶፍትዌር ልማት እና ስሪት ቁጥጥር ጊትን በመጠቀም የሚያስተናግድ ድር ላይ የተመሰረተ መድረክ ነው ። ታዳጊዎች በፕሮጀክቶች ላይ በጋራ እንዲሰሩ ፣ ኮድ እንዲጋሩ ፣ እና የሶፍትዌር ልማት የስራ ፍሰት እንዲያስተዳድሩ ትብብር ያደርጋል ።

ጊትና ጊትሩብን ማደራጀት
ጊት እና ጊትሁብን ማቀናጀት መጀመሪያ ላይ በጣም ውስብስብ ሊመስል ይችላል ፣ ነገር ግን በትብብር እና በኮድ አያያዝ ውጤታማ ለመሆን ወሳኝ እርምጃ ነው ። በደረጃ መሪው እነሆ ፥

1. የማውረድ ኃይል ፦ 

ጊትን ለማውረድ የቀረበውን አገናኝ ይጎብኙ ። በመኪናዎ ላይ ይጫኑት ።

2. የጊታር መለያ ፍጠር ፦

የጌትሑብ መለያ ከሌለዎት ፥ የጊትሆብን ድረ ገጽ ይጎብኙና አንድ ይፍጠሩ ። ቀጥተኛ ሂደት ነው ።

3. ከጊት ጋር አገናኝ መለያ ፦ 

እንደ ስርዓተ ክወናዎ መሰረት ፣ ተርሚናል በMac / ሊንከን ወይም Git Bash ላይ Windows ላይ ይጠቀሙ ። እነዚህ የጽሁፍ ትእዛዞችን በመጠቀም ከኮምፒውተርህ ጋር ለመገናኘት የሚያስችሉህ የትእዛዝ መስመር ግንኙነቶች ናቸው ። የgትሑብ መለያዎን ከጊት ጋር ለማገናኘት የተዘጋጁ ትእዛዞችን ያስገቡ ።

4. የ ኤስ.ኤስ.ኤ ቁልፍን ማዘጋጀት (ጥሩ ቢሆንም የሚመከር ቁልፍ) ፦

የኤስ ኤስ (ሚስጥር ሼል) ቁልፎች ከጊትሁብ መለያዎ ጋር ለመገናኘት ደህንነቱ የተጠበቀ መንገድ ይሰጣሉ ። አሁን ያሉት የኤስ ኤስ ፒ ኬ ቁልፎች ካሉህ ፈትሽ ፤ አስፈላጊ ከሆነም አዳዲስ የኤስ ኤስ ቁልፎችን ማፍለቅ ትችላለህ ። ለጊትሑብ የኤስ ኤስ ቁልፍህን እንድትጨምር የተሰጠውን ትእዛዝ ተከተል ።
