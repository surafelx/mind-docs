

## CSS

ሲኤስኤስ (የሽባ ቅጥ ሸሌቶች) የሰዓሊው ካሴት ሲሆን ቅጥንና እይታን ይጨምራል
ከ HTML ጋር ወደተፈጠረው የድር ገጾች ይግባኝ በል ። አቀማመጥን እንድትቆጣጠር ያስችልሃል ፡ ፡
የይዘትህ ቀለሞች ፣ ቁምፊዎች እና አጠቃላይ አቀራረብ ።

### Introduction to CSS:

Css የሚንቀሳቀሰው HTML አባሎችን በመምረጥ እና ቅጾችን በመተግበር ነው ። እንስማ
ቀላል CSS ምሳሌን ይመልከቱ ፥

```css filename="styles.css"
body {
  font-family: 'Arial', sans-serif;
  background-color: #f0f0f0;
}

h1 {
  color: #333;
}

p {
  font-size: 16px;
  line-height: 1.5;
}
```

- `body`: የ HTML ሰነድ ሙሉ አካልን ይምረጣል ።
- `h1`, `p`: የተወሰኑ የHTML ንጥረ ነገሮችን (ርእሶች እና አንቀጾች) ይምረጡ ።
- `font-family`, `background-color`, `color`, `font-size`, `line-height`:
የተለያዩ የሲ.ኤስ.አይ.ኤስ ንብረቶች ለደንበኝነት ።
