## JavaScript

JavaScript ከድር ልማት በስተጀርባ የሚገኝና እርስ በርስ ግንኙነት ለመፍጠር የሚያስችል ተለዋዋጭ ኃይል ነው
ልግስናን ፤ Html እና CSSንንም በተመሳሳይ ጊዜ ውስጥ ለማሰስ ይፈቅድልሃል ፣
የተጠቃሚ ድርጊቶች ምላሽ በመስጠት እና የበለጠ የሚሳተፍ የተጠቃሚ ልምድ በመፍጠር ።

# # የ JavaScript አጠቃላይ እይታ

Javascript በቀጥታ ወደ HTML ወይም በተለያዩ ፋይሎች ውስጥ ሊካተት ይችላል ።
እዚህ ላይ የጃቫ ስክሪን መግለጫ በተግባር ላይ ነው ፥

```js filename="scripts.js"
// JavaScript in HTML
<script>
    // Function to display a greeting
    function greet() {
        alert('Hello, welcome to our website!');
    }
</script>

// JavaScript in an external file (example.js)
// Function to change the background color
function changeColor() {
    document.body.style.backgroundColor = 'lightblue';
}
```

- `function greet()`: ሰላምታ አሰጣጥን በተመለከተ ጃቫ ስክሪፕት ተግባር ይገልጣል ።
- `function changeColor()`: የበስተጀርባውን ቀለም ለመቀየር ተግባርን ይገልጸዋል ።
