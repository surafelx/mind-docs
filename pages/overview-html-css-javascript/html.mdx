# HTML አጭር ማብራሪያ

ኤችቲኤምኤል የሕንፃ ማዕቀፍ ነው ፣ አወቃቀርና አቀማመጥን እንመርምር
ይዘት ። መሰረታዊ የHTML አወቃቀርን እንመርምር ፥

```html filename="index.html"
<!doctype html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>This is a simple webpage built with HTML.</p>
    <img src="example.jpg" alt="An example image" />
    <a href="https://www.example.com">Visit Example.com</a>
  </body>
</html>
```

- `<!DOCTYPE html>`: የHTML የሰነድ አይነት እና ስሪት ይገልጻል ።
- `<html>`: የHTML ገጽ ሥሩ አላባ ።
- `<head>`:  ስለ HTML ሰነድ የብረታ-መረጃ ይዟል ።
- `<title>`: : የድርገጽ ርእስ አዘጋጅ (በአሳሽ ትር ላይ የታየ) ።
- `<body>`: የድርገጹ ይዘት ይዟል ።
- `<h1>`, `<p>`, `<img>`, `<a>`: የተለያዩ የHTML ታጎችን ለራስጌዎች ፣ ለአንቀጽ ፣
ምስሎች እና አገናኞች ።
