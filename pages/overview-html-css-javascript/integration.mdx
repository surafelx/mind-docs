# HTML ፣ CSS እና JavaScript ውህደት


እነዚህ ሦስት ቴክኖሎጂዎች እርስ በርስ ተቀናጅተው በመሥራት ተለዋዋጭ የሆነ የድር ተሞክሮ ይፈጥራሉ ።
Html ይዘቱን መዋቅሮች ፣ CSS ቅጾች ፣ እና JavaScript ያክላሉ
ተግባር ። አንድ አዝራር ጠቅ ሲደረግ ፣ ለውጦቹን የያዘውን ሁኔታ ይመልከቱ
የድር ገጽ የበስተጀርባ ቀለም ። ይህ ሦስት ቴክኖሎጂዎች መሥራትን ይጨምራል
አንድ ላይ ፥

```html filename="index.html"
<!DOCTYPE html>
<html>
  <head>
    <title>Dynamic Web Experience</title>
    <style>
      body {
        font-family: 'Arial', sans-serif;
        background-color: #f0f0f0;
      }

      button {
        padding: 10px;
        font-size: 16px;
      }
    </style>
    <script>
      function changeColor() {
        document.body.style.backgroundColor = 'lightblue'
      }
    </script>
  </head>
  <body>
    <h1>Welcome to a Dynamic Web Experience</h1>
    <button onclick="changeColor()">Change Background Color</button>
  </body>
</html>
```

- `HTML` የገጹን ይዘት መዋቅሩ ።
- `CSS` ሰውነት እና የቁልፍ አካላት ።
- `JavaScript` አዝራር በሚኖርበት ጊዜ ከበስተጀርባው ያለውን ቀለም ለመቀየር እንቅስቃሴን ይጨምራል
ጠቅ ተደርጓል ።

HTML ፣ CSS ፣ እና JavaScript እንዴት በጋራ መስራት እንደሚችሉ መረዳት ለመሞከር ቁልፍ ነው
ማሰማራት እና የተጠቃሚ-ወዳጃዊ ድር ጣቢያዎችን ። በዚህ መንገድ እየተጓዝክ ስትሄድ ፣
ወደ እያንዳንዱ ቴክኖሎጂ ጠለቅ ብሎ የሚገባ ሲሆን ተለዋዋጭና ሞቅ ያለ የፈጠራ ችሎታ ማዳበር
የሚያጓጓ የድር ተሞክሮዎች ። ደስተኛ ኮድ!
