## የኋለኛ ዌብ ደቨሎፕመንት ማስተዋወቅ

ከእያንዳንዱ የድር መተግበሪያ ጀርባ አስማቱ በሚከናወንበት ወደ ኋላ መለስ ልማት አለም እንኳን ደህና መጡ ። በዚህ ርዕስ ውስጥ ፣ የጀርባ ዕድገትን ዓለም ለማደፍረስ ፣ የድርጅቱን ሚና ፣ ቁልፍ ጽንሰ ሐሳቦችንና ከድር አገልጋይነት ጎን ተሰልፈው የሚሠሩትን አስፈላጊ ቴክኖሎጂዎች ለመመርመር ጉዞ እንጀምራለን ።

### የኋለኛ ዌብ ደቨሎፕመንት  ምንድን ነው?
የኋለኛ ዌብ ደቨሎፕመንት የድር ልማት አገልጋይነትን የሚያመለክት ሲሆን ድረ ገጽ ወይም የድር መተግበሪያ ተግባር በሚያደርገው የሎጅክ ፣ የውሂብ ጎታዎች እና የአገልጋይ ክንውኖች ላይ ያተኩራል ። የፊት ለፊት ልማት ሥራው ተጠቃሚዎች በቀጥታ በሚገናኙበት ጉዳይ ላይ የሚያተኩር ቢሆንም በጀርባው ኋላ በኩል ያለው ሂደት ግንባሩ ያለምንም ችግር እንዲንቀሳቀስ ያስችለዋል ።

### የኋለኛ ዌብ ደቨሎፕመንት  ቁልፍ አስተሳሰቦች 
#### አገልጋይ
በጀማሪ ዕድገት ውስጥ አገልጋዩ ነው ። አገልጋይ ኮምፒዩተር ወይም ደንበኛው መሳሪያዎች ለሚያቀርቡላቸው ጥያቄዎች (እንደ አሳሾች ወይም የሞባይል መተግበሪያዎች) በአውታረ መረብ ላይ ምላሽ የሚሰጥ ስርዓት ነው ። አገልጋዮች እነዚህን ጥያቄዎች ያዘጋጃሉ ፣ አስፈላጊ የሆኑ ኮምፖዛሎችን ያከናውናሉ እንዲሁም ምላሾችን ይመልሳሉ ።

#### ውሂብ ጎታ
የኋለኛ ዌብ ደቨሎፕመንት ብዙውን ጊዜ ውሂብን ለማከማቸት ፣ መልሶ ለማግኘትና ለማስተዳደር ዳታ ቤዝ ቦዮችን በመጠቀም ይሠራል ። እንደ የተጠቃሚ ውሂብ ፣ ይዘት እና የመተግበሪያ ውቅር ያሉ መረጃዎችን ያለማቋረጥ ለማከማቸት የውሂብ ጎታዎች ወሳኝ ናቸው ።

#### API
አፒስ በተለያዩ የሶፍትዌር ሥርዓቶች መካከል የሐሳብ ልውውጥ እንዲኖር ያስችላል ። አፒስ እድገትን በማስቀደም ማመልከቻዎቹ እንዴት ሊገናኙ እንደሚገባ የሚገልጽ ደንብ አፒስ አወጣ ። የፊት ለፊት ክፍሎቹ ውሂቡን እንዲጠይቁትና ከጀርባው ያለውን መረጃ በተቀናጀ መንገድ እንዲቀበሉ ይፈቅዳሉ ።

#### የቋንቋዎች እና ማዕቀፎች
ጀርባን የሚያዳብሩ ሰዎች እንደ ዘንዶ ፣ ጃቫ ስክሪፕት (ኖድ.ጄስ) ፣ ሩቢ ፣ ጃቫ ወይም ፒፕ ያሉ የፕሮግራም ቋንቋዎች ይጠቀማሉ ። እንደ ጃጃንጎ (Python) ፣ ኤክስፕረስ (ጃቫScript) ወይም ሩቢ (ሩቢ) የመሳሰሉ የህግ ማዕቀፎች የኋለኛ ኮድን ለመገንባትና ለማደራጀት የተቀናበረ መንገድ ይሰጣሉ ።

#### ማረጋገጫ እና ፈቃድ
የመረጃና የምግባረ ዕድሎችን አስተማማኝ ማድረግ የጀርባ ዕድገት ወሳኝ ገጽታ ነው ። የተጠቃሚዎችን ማንነት ማረጋገጥ ፣ ፈቃድ መስጠት ግን ምን ድርጊቶች እንዲፈጸሙ እንደተፈቀደላቸው ይቆጣጠራል ፡ ፡

#### Server-Side
Ssrን በአገልጋዩ ላይ የድር ገጾችን ከመላክ በፊት በአፈጻጸም ላይ በማብቃት ደንበኛው ዘንድ መላክን ያካትታል ። አፒስ ተለዋዋጭ እና በይነተገናኝ የድር መተግበሪያዎችን በማብቃት የፊት ለፊት እና የመልሶ ማስቀመጫ ውሂብን በብቃት እንዲለዋወጡ ይፈቅዳል ።

#### የኋለኛ ዌብ ደቨሎፕመንት  ሂደት
#### ጥያቄ ትንታኔ
በጀማሪው ሊይዛቸው የሚገቡ የፕሮጀክት መስፈርቶችን ፣ ምሰራራዎችን ፣ እና የውሂብ መዋቅሮችን ተረዳ ።

#### የውሂብ ጎታ ንድፍ
የውሂብ ጎታ ውሂብ እንዴት እንደሚደራጅ ፣ እንደሚከማች እና እንደሚያድግም የሚያሳይ የውሂብ ጎታ ውቅር ይንደፉ ።

#### Server Side Logic
የሚገቡ ጥያቄዎችን ፣ የውሂብ ሂደትን ፣ እና ተገቢውን ምላሾችን ለማመንጨት የአገልጋይ-ወገን ምክንያታዊ ምስሎችን ይጻፉ ።

#### API ልማት
የፊት ለፊት ተፋላሚው ከጀርባው ጋር እንዲነጋገር የሚፈቅድ አፒስ ያዘጋጁ ፤ ይህም እንዴት ውሂብ መጠየቅ እና መላክ እንዳለበት ያትታሉ ።

#### የደህንነት ስራ
የተጠቃሚ ማረጋገጫ ፣ ምስጠራ ፣ እና የመዳረሻ መቆጣጠሪያዎችን ጨምሮ የደህንነት እርምጃዎችን ይተገብራል ።

#### ፈተና
የመልሶ ማቋቋም ተግባሩ በሚጠበቀው መልኩ እንዲከናወን ፣ የተለያዩ ሁኔታዎችንና ጠርዝ ጉዳዮችን እንዲይዝ የተሟላ ምርመራ ያካሂዳል ።

#### ውድቂያ
የመልቀቂያ ኮዱን እና የውሂብ ጎታውን ወደ አገልጋዩ ወይም ወደ ደመና መድረክ በማሰማራት መተግበሪያውን ለተጠቃሚዎች ተደራሽ ያደርጋል ።

#### የመመረቂያ እና ጥገና
የጀርባውን አፈፃፀም ፣ የአድራሻ ጉዳዮችን ያለማቋረጥ መከታተል እና እንደአስፈላጊነቱ ማሻሻያዎችን ማድረግ ።

# ተወዳጅ የኋለኛ ዌብ ደቨሎፕመንት  ቴክኖሎጂዎች

#### Node.js

ታዳጊዎች በአገልጋይ በኩል JavaScript እንዲጠቀሙ የሚፈቅድ የጃቫ ስክሪን ሰዓት ። በተለይ በቀላሉ የሚሰራጭ እና ከፍተኛ አፈጻጸም ያላቸውን የድር መተግበሪያዎች ለመገንባት ተመራጭ ነው ።

#### Django (Python)

ፈጣን ልማትን የሚያበረታታ ከፍተኛ ደረጃ ያለው የፓይታኖን ድር ማዕቀፍ ሲሆን አራታዊ ንድፍ አለው ።

#### Ruby on Rails

የአውራጃ ስብሰባን በማጉላት እና ታዳጊዎች የዘረፋ ትግበራዎችን በፍጥነት እንዲገነቡ በማስቻል በሩቢ የተቀረጸ የድር መተግበሪያ ማዕቀፍ ነው ።

#### Express.js

አነስተኛ እና ተለዋዋጭ ኖድ.ጂስ የድር መተግበሪያ ማዕቀፍ የድር እና ሞባይል መተግበሪያዎችን ለማልማት ሰፊ የገጸ ባህሪያትን ይሰጣል ።

#### ትእምርተ ነጥብ

ምትኬ ልማት የድር መተግበሪያዎች የጀርባ አጥንት ነው ፣ በተግባር ፣ በውሂብ አስተዳደር እና ደህንነት ኃይል በመስጠት ። ወደኋላ እየተመለስክ ባለህበት ዓለም ውስጥ ምርምር ለማድረግ የሚያስችሉ ቴክኖሎጂዎችና ጽንሰ ሐሳቦች በብዛት ይገኛሉ ። የጀርባ አጥንት ለመሆን ወይም ስለድር ልማት ቅድስና ያለውን ግንዛቤ ለማግኘት ብትፈልግ ፣ የጀርባውን ጠንቅቀህ ማወቅ ፣ ጠንካራ እና ተለዋዋጭ ዲጂታል ልምዶችን ለመገንባት ቁልፍ እርምጃ ነው ። ደስተኛ ኮድ!
